# SAÉ 23 : Mettre en place une solution informatique d'entreprise (Web & BDD)

## 🎯 Objectif du projet
Concevoir et développer une application web dynamique reliée à une base de données relationnelle pour stocker, mettre à jour et afficher des informations.

## 🛠️ Technologies et outils utilisés
* Langages de programmation (HTML5, CSS3, PHP)
* Système de Gestion de Base de Données (MySQL / MariaDB)
* Serveur web Apache

## 🚀 Les défis rencontrés & Solutions
* **Défi :** Assurer la communication fluide et sécurisée entre le formulaire web et les tables de la base de données.
* **Solution :** Utilisation de requêtes SQL préparées pour sécuriser les injections et structuration rigoureuse du schéma relationnel.

## 📈 Résultats obtenus et compétences acquises
* Mise en ligne d'une plateforme web fonctionnelle permettant d'administrer les données en temps réel.
